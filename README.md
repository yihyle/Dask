# Dask - 자유롭고 안전한 소통 공간

Next.js 15와 TypeScript를 기반으로 한 현대적인 게시판 애플리케이션입니다.

## 🚀 주요 기능

- 익명 게시판
- 반응형 디자인
- 실시간 검색
- 카테고리 분류
- 정렬 기능
- 페이지네이션

## 🛠️ 기술 스택

- Next.js 15
- TypeScript
- Tailwind CSS
- React 19

## 🚀 시작하기

```bash
npm install
npm run dev
```

## 📁 프로젝트 구조

- `app/` - Next.js App Router
- `components/` - 재사용 가능한 컴포넌트
- `types/` - TypeScript 타입 정의
- `constants/` - 상수 정의
- `utils/` - 유틸리티 함수
